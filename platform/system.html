平台系统的数据指标详解

指标 具体含义
system.cpu.guest 运行虚拟处理器时花费时间的百分比,仅适用于虚拟机
system.cpu.idle 当前 CPU 空闲百分比
system.cpu.interrupt 进程用于处理中断所花费时间的百分比
system.cpu.iowait 当前 CPU 处在等待 IO 操作的百分比
system.cpu.stolen 虚拟 CPU 花在等待管理程序分配其他虚拟 CPU 的时间百分比,仅适用于虚拟机
system.cpu.system 当前 CPU 运行内核所占百分比
system.cpu.user 当前 CPU 运行用户进程的百分比
system.disk.free 空余磁盘空间 （byte）
system.disk.in_use 已使用磁盘占比
system.disk.read_time_pct 在磁盘读取上花费时间的百分比
system.disk.total 磁盘总空间 （byte）
system.disk.used 已使用的磁盘空间 （byte）
system.disk.write_time_pct 在磁盘写入上花费时间的百分比
system.fs.inodes.free 空闲 inode 数
system.fs.inodes.in_use 已使用 inode 占比
system.fs.inodes.total inode 总数
system.fs.inodes.used 已使用 inode 数
system.io.avg_q_sz 向设备发出的请求的平均队列大小
system.io.await 服务向设备发送 IO 请求平均数，包括花在队列中的时间 （millisecond）
system.io.r_s 设备每秒的读请求的数
system.io.rkb_s 设备每秒读的 kibibytes 的数量
system.io.util 向设备请求 IO 时 CPU 占比
system.io.w_s 设备每秒的写请求的数
system.io.wkb_s 设备每秒写的 kibibytes 的数量
system.load.1 过去 1 分钟内系统负载
system.load.5 过去 5 分钟内系统负载
system.load.15 过去 15 分钟内系统负载
system.load.norm.1 过去一分钟系统负载（每单核 CPU）
system.load.norm.5 过去五分钟系统负载（每单核 CPU）
system.load.norm.15 过去十五分钟系统负载（每单核 CPU）
system.mem.buffered 系统分配却未被使用的 buffer 数量
system.mem.cached 系统分配却未被使用的 cache 数量
system.mem.committed 存储在已被保留的磁盘分页上的物理内存量
system.mem.free 可用内存量 （byte）
system.mem.used 已用内存量 （byte）
system.mem.nonpaged 操作系统用于不能被写入磁盘，但必须保持在物理内存中的对象的物理内存量
system.mem.page_free 可用的 page file 数量
system.mem.page_pct_free 使用中的 page file 数量占总量的比例
system.mem.page_total page file 的总量
system.mem.page_used 使用中的 page file 数量
system.mem.paged 操作系统用于不用时可以被写入磁盘的对象的物理内存量
system.mem.pct_usable 内存使用百分比
system.mem.shared 共享内存的大小
system.mem.total 物理内存总大小
system.mem.usable 内存 free buffered cached 的总和
system.net.bytes_rcvd 设备每秒接收到的字节数
system.net.bytes_sent 每秒发送的字节数
system.net.packets_in.count 接口接收的数据包的数目
system.net.packets_in.error 设备驱动器检测到的接收错误包的数量
system.net.packets_out.count 接口发送的数据包的数目
system.net.packets_out.error 设备驱动器检测到的发送错误包的数量
system.net.tcp.in_segs 接收的 TCP 段的数量
system.net.tcp.out_segs 发送的 TCP 段的数量
system.net.tcp.retrans_segs 重发的 TCP 数据包的数量
system.net.udp.in_datagrams 接收的 UDP 数据包
system.net.udp.in_errors 接收的错误 UDP 数据包
system.swap.free 空余交换空间
system.swap.pct_free Swap 已使用百分比
system.swap.total Swap 总大小
system.swap.used 交换空间总量
system.swap.swapped_in 每秒交换回内存的总量 / bytes
system.swap.swapped_out 每秒交换出内存的总量 / bytes
