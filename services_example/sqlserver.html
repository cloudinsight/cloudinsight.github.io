SQL Server

Microsoft SQL Server 是 Microsoft 所推出的关系数据库解决方案，具有使用方便可伸缩性好与相关软件集成程度高等优点。

Cloudinsight 通过授权收集 SQL Server 数据，可视化 SQL Server 性能。

性能指标

Cloudinsight 采集 SQL Server 以下性能指标：

指标 具体含义
sqlserver.access.page_splits 每秒数据页分割数量
sqlserver.buffer.cachehitratio 缓冲区缓存命中率
sqlserver.buffer.checkpoint_pages 每秒数据库脏页写入磁盘数量，"脏页"是已输入缓存区高速缓存已修改但尚未写入磁盘的数据页
sqlserver.buffer.pagelifeexpectancy 一个页面驻留在缓冲池的持续时间/s
sqlserver.stats.batch_requests 每秒批量请求数
sqlserver.stats.connections 用户连接数
sqlserver.stats.lock_waits 等待锁的进程时间
sqlserver.stats.procs_blocked 进程阻塞的数量
sqlserver.stats.sql_compilations 每秒SQL编译数
sqlserver.stats.sql_recompilations 每秒SQL重新编译数

配置 SQL Server 监控

1. 确保 SQL Server 启用了混合模式（Windows 身份验证和 SQL Server 身份验证）验证

2. 创建一个只读用户，探针用此用户连接到 SQL Server 

CREATE LOGIN oneapm WITH PASSWORD = '<您的密码>';
CREATE USER oneapm FOR LOGIN oneapm;
GRANT SELECT on sys.dm_os_performance_counters to oneapm;
GRANT VIEW SERVER STATE to oneapm;

3. 配置探针连接到 SQL Server

通过探针管理器配置 SQL Server 如下信息：

instances:
  -  host: MY_HOST,MY_PORT
     username: oneapm
     password: '<您的密码>'

4. 重启探针

通过探针管理器重启探针 Actions，restart。

5. 确认状态

在探针管理器中查看信息栏 Log and Status 下面的 Agent Status，如果有以下信息的输出，代表配置正确。

Checks
======

  [...]

  sqlserver
  ---------
      - instance #0 [OK]
      - Collected 8 metrics & 0 events

有关 Agent Info 信息的查看，请访问帮助中心，查看 Cloudinsight Agent 常用操作。
